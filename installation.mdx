## Installation rapide

<CodeGroup>
  ```bash npm
  npm i @ianord/table --registry=https://npm.ianord.fr
  ```

  ```bash pnpm
  pnpm i @ianord/table --registry=https://npm.ianord.fr
  ```

  ```bash yarn
  yarn add @ianord/table --registry=https://npm.ianord.fr
  ```
</CodeGroup>

`@ianord/table` est installé.

## Installation avec mise en place du registry

Pour pouvoir installer et mettre à jour le package sans mentionner le registry, vous pouvez simplement créer un fichier `.npmrc` à la racine du projet

```dockerfile .npmrc
registry=https://npm.ianord.fr
```

Puis installez simplement `@ianord/table`

<CodeGroup>
  ```bash npm
  npm i @ianord/table
  ```

  ```bash pnpm
  pnpm i @ianord/table
  ```

  ```bash yarn
  yarn add @ianord/table
  ```
</CodeGroup>

`@ianord/table` est installé, et vous pouvez désormais mettre à jour le package sans spécifier à nouveau le registry.
