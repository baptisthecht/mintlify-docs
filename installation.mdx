<Info>**Prerequisite**: You must have a React project initialized.</Info>

Step 1. Install @ianord/table on your project:

<CodeGroup>

```bash npm
npm i @ianord/table --registry=https://npm.ianord.fr/
```

```bash pnpm
pnpm i @ianord/table --registry=https://npm.ianord.fr/
```

```bash yarn
yarn add @ianord/table --registry=https://npm.ianord.fr/
```

</CodeGroup>

If you dont want to specify the registry at each update, you can just create a `.npmrc` file :

```dockerfile .npmrc
registry=https://npm.ianord.fr/
```

Then you can run

<CodeGroup>

```bash npm
npm i @ianord/table
```

```bash pnpm
pnpm i @ianord/table
```

```bash yarn
yarn add @ianord/table
```

</CodeGroup>

or

<CodeGroup>

```bash npm
npm update @ianord/table
```

```bash pnpm
pnpm update @ianord/table
```

```bash yarn
yarn update @ianord/table
```

</CodeGroup>

Step 2. Import the styles.css file at the root of your project

```typescript App.tsx
import "@ianord/table/styles.css";
```

Your project can now use `@ianord/table`.
