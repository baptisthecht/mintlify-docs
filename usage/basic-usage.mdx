### Create table instance

To create a table instance, use the `useAutoTable` function from `@ianord/table` package. The `useAutoTable` function accepts two arguments:

<ResponseField name="data" type="T[]" required>
  Data to be displayed in the table. This is an array of objects.
</ResponseField>
<ResponseField name="config" type="ATConfig<T>">
  The table configuration object. See the [configuration](/configuration) guide for more details.
</ResponseField>

```tsx app/page.tsx
const data = await fetch("...");
const config = {
	actions: {
		onClick: (action, row) => {
			console.log(action, row);
		},
	}
	pagination: {
		firstAndLastPageButtons: true,
	}
}

const instance = useAutoTable(data, config);
```

### Rendering table

To render the table, use the `AutoTable` component from `@ianord/table` package. The `AutoTable` component accepts the table instance as a prop.

```tsx app/page.tsx
<AutoTable instance={instance} />
```

### Full code

```tsx app/page.tsx
import { useAutoTable, AutoTable } from "@ianord/table";

export default function Page() {
	const { data, loading } = await useQuery("...");
	const config = {
		loading,
		actions: {
			onClick: (action, row) => {
				console.log(action, row);
			},
		}
		pagination: {
			firstAndLastPageButtons: true,
		}
	}

	const instance = useAutoTable(data, config);

	return <AutoTable instance={instance} />;
}
```
